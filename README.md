# 帅舅舅 - 英语数学学习助手

一个专门为高一学生设计的英语和数学学习 Android App。

## 功能

- 📖 **英语单词**：初中词汇 + 高中词汇，含音标和详细解析
- 📝 **单词测验**：选择题模式，自动收录错题
- 🧮 **数学练习**：初中数学 + 高中数学，含详细解题步骤
- 📚 **错题本**：自动收集错题，方便复习
- ✅ **每日打卡**：坚持学习，养成习惯
- 📊 **学习统计**：查看学习进度和成果

## 技术栈

- Flutter 3.16+
- Dart 3.0+
- SharedPreferences（本地数据存储）

## 构建方法

```bash
flutter pub get
flutter build apk --release
```

APK 文件位置：`build/app/outputs/flutter-apk/app-release.apk`
